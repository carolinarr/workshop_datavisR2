Does date time occur in the am or pm?

Description

Does date time occur in the am or pm?

Usage

am(x)

pm(x)

Arguments

x

a date-time object

Value

TRUE or FALSE depending on whether x occurs in the am or pm

Examples

x <- ymd("2012-03-26")
am(x)
pm(x)
